# Web Scraping a un artículo de Clarín 

En el momento en que finalizó el partido entre River y Gimnasia de Mendoza, se realiza un web scraping a la página de Clarín 
para leer lo que dejó el gran choque entre ambos equipos. 

Se utiliza las librerias bs4 Beautiful Soup y se recorren las listas para almacenarlas y generar cadenas de textos.
